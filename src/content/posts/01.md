当你在 Vercel 上部署了网站后，默认会获得一个 `your-project.vercel.app` 域名。但为了打造专业形象，你通常会希望使用自己的自定义域名。本文将详细介绍如何在 Vercel 上配置自定义域名，从 DNS 设置到域名验证的完整流程。

### 准备工作

在开始之前，确保你已经：

1. 在 Vercel 上成功部署了你的项目
2. 拥有一个已注册的自定义域名（可在阿里云、腾讯云 等域名注册商处购买）
3. 能够访问域名注册商的 DNS 管理控制台

### 第一步：在 Vercel 中添加自定义域名

1. 登录你的 Vercel 账户并进入项目控制台
3. 选择左侧菜单中的「Domains」选项
4. 点击右上角的「Add Domain」按钮

![13fd3d5e-467a-9e44-144e-dc228f9eddd3.png](https://raw.githubusercontent.com/sxy15/bbb/main/assets/13fd3d5e-467a-9e44-144e-dc228f9eddd3.png)

在弹出的对话框中，输入你的自定义域名（例如 `zzudesign.site`），然后点击「Save」按钮。

![8897e7c7-3533-e21a-9a5d-9cc9974a5fc5.png](https://raw.githubusercontent.com/sxy15/bbb/main/assets/8897e7c7-3533-e21a-9a5d-9cc9974a5fc5.png)

### 第二步：配置 DNS 记录

添加域名后，Vercel 会提供需要配置的 DNS 记录。根据你的域名是根域名还是子域名，配置方式略有不同。

#### 配置根域名（如 example.com）

对于根域名，你需要添加一个 A 记录，指向 Vercel 提供的 IP 地址。

1. 登录你的域名注册商的管理控制台
2. 找到 DNS 管理或域名解析设置
3. 添加一条 A 记录：
   - 记录类型：A
   - 记录值/主机名：`@`
   - 记录值/目标：`xx.xx.xxx.xx`

#### 配置子域名（如 www.example.com）

对于子域名（如 `www.zzudesign.site`），你需要添加一条 CNAME 记录，指向你的 Vercel 分配域名。

1. 在 DNS 管理界面中添加一条 CNAME 记录：
   - 记录类型：CNAME
   - 记录值/主机名：`www`
   - 记录值/目标：你的 Vercel 项目域名（如 `zzu-design-docs.vercel.app`）

![830e85f6-cdcf-d51d-c09d-a698b1eb26a5.png](https://raw.githubusercontent.com/sxy15/bbb/main/assets/830e85f6-cdcf-d51d-c09d-a698b1eb26a5.png)

### 第三步：配置域名重定向（可选）

为了让用户无论访问 `example.com` 还是 `www.example.com` 都能正常访问你的网站，你可以在 Vercel 中设置域名重定向。

1. 在 Vercel 的 Domains 页面中，点击你已添加的域名右侧的「Edit」按钮
2. 选择「Redirect to Another Domain」选项
3. 选择重定向类型（通常选择 307 Temporary Redirect 或 308 Permanent Redirect）
4. 输入目标域名（如从 `zzudesign.site` 重定向到 `www.zzudesign.site`）
5. 点击「Save」保存设置

### 第四步：验证域名配置

配置完 DNS 记录后，Vercel 会自动验证你的域名配置。这个过程可能需要一些时间，因为 DNS 记录的传播通常需要几分钟到几小时不等。

在验证过程中，你可能会看到以下状态：

- **Invalid Configuration**：DNS 记录尚未正确传播或配置不正确
- **Generating SSL Certificate**：域名验证已通过，正在生成 SSL 证书（Vercel 会自动为你的域名配置免费的 SSL 证书）
- **Valid Configuration**：配置已成功，域名已激活

### 最佳实践

1. **同时配置根域名和 www 子域名**：为了提供更好的用户体验，建议同时配置两者
2. **设置重定向**：选择一个主要域名（通常是 www 子域名），将另一个域名重定向到它
3. **定期检查 DNS 记录**：Vercel 可能会更新推荐的 IP 地址，定期检查确保配置保持最新
4. **使用 HTTPS**：Vercel 自动提供 SSL 证书，确保你的网站通过 HTTPS 访问

### 总结

通过本文介绍的步骤，你可以轻松地在 Vercel 上配置自定义域名，提升你的网站的专业性和品牌形象。整个过程主要包括在 Vercel 中添加域名、配置 DNS 记录、设置域名重定向以及验证域名配置。虽然 DNS 记录的传播可能需要一些时间，但一旦完成，你就可以通过自己的自定义域名访问部署在 Vercel 上的网站了。

如果在配置过程中遇到任何问题，建议查阅 Vercel 的官方文档或联系他们的支持团队获取帮助。
